	# Proj0049: Prefer the latest released language version
	To benefit from the latest language features, the latest released version of
	the language (if it is C#, or VB.NET) should be used.


	| .NET |   C# | VB.NET |
	|-----:|-----:|-------:|
	|  6.0 | 10.0 |  16.9  |
	|  7.0 | 11.0 |  16.9  |
	|  8.0 | 12.0 |  16.9  |
	|  9.0 | 13.0 |  17.13 |
	| 10.0 | 14.0 |  17.13 |
