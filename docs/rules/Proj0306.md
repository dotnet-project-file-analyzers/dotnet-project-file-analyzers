# Proj0306: Avoid local package sources
Package sources should be verifiable and reliable. A local package source is
most likely nown of those.

## Non-compliant
``` xml
<configuration>

  <packageSources>
    <clear />
    <add key="nuget.org" value="C:/NuGet/packages" />
  </packageSources>

</configuration>
```

## Compliant
``` xml
<configuration>

  <packageSources>
    <clear />
    <add key="nuget.org" value="https://api.nuget.org/v3/index.json" />
  </packageSources>

</configuration>
```
