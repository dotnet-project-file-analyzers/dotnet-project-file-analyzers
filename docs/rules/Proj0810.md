# Proj0810: Remove unused package versions
When using [Central Package Management](Proj0800.md) the central
`Directory.Packages.props` contains `<PackageVersion>` elements for all
`<PackageReference>` elements in the solution. To reduce noise, this rule reports on
`<PackageVersion>` elements that are (no longer) referenced. This rule requires a
[`.net.csproj` SDK project](../general/sdk.md) to work.
