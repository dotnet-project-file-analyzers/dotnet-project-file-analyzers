# Proj0811: Directory.Build.props and Directory.Packages.props should be in the same directory
Both `Directory.Build.props` and `Directory.Packages.props` allow sharing MSBuild
configuration across multiple project files. To prevent confusion, when using both
(the latter is only used when [Central Package Management](Proj0800.md)
has been enabled), they should be put in the same directory.
