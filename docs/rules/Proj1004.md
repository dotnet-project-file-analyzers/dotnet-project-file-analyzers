# Proj1004: .NET Project File Analyzers SDK not longer requires separate package
With the release of [DotNetProjectFile.Analyzers v1.12.0](https://www.nuget.org/packages/DotNetProjectFile.Analyzers/1.12.0)
the `.net.csproj` works without including the [DotNetProjectFile.Analyzers.Sdk](https://www.nuget.org/packages/DotNetProjectFile.Analyzers.Sdk)
package. More info on how the `.net.csproj` work can be read [here](../general/sdk.mk).

## Non-compliant
``` xml
<Project Sdk="Microsoft.NET.Sdk">

  <ItemGroup>
    <PackageReference PrivateAssets="all" Include="DotNetProjectFile.Analyzers" Version="1.11.0" />
  </ItemGroup>

</Project>
```

## Compliant
``` xml
<Project Sdk="Microsoft.NET.Sdk" />
```
