# Proj1005: Suppress NuGet issues per package
The NuGet warnings [NU1701](https://learn.microsoft.com/nuget/reference/errors-and-warnings/nu1701),
[NU1702](https://learn.microsoft.com/nuget/reference/errors-and-warnings/nu1702), and
[NU1703](https://learn.microsoft.com/nuget/reference/errors-and-warnings/nu1703)
report issues where the fallback target might not be compatible with the
selected `<TargetFramework>`.

As the raised issues are reported on a specific dependency, it is best to
suppress it on that specific dependency only (assuming it is safe to do so).

## Non-compliant
``` xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
    <NoWarn>NU1701</NoWarn>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="LegacyPackage" Version="1.0.0" />
  </ItemGroup>

</Project>
```
## Compliant
``` xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="LegacyPackage" Version="1.0.0" NoWarn="NU1701" />
  </ItemGroup>

</Project>
```
