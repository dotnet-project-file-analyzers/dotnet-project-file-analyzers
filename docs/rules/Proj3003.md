# Proj3003: Trim whitespace 
Leading or trailing isignificant whitespace can cause syntax errors or making
the comparing of file changes harder. This rule advises to remove leading and
trailing whitespace when possible.

## Non-compliant
```
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="DotNetProjectFile.Analyzers " Version = "1.8.0" PrivateAssets="all" />
  </ItemGroup>

</Project>
```

## Compliant
```
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="DotNetProjectFile.Analyzers" Version="1.8.0" PrivateAssets="all" />
  </ItemGroup>

</Project>
```
