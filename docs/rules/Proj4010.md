# Proj4010: Sections should contain at least one key-value pair
An [INI file](https://en.wikipedia.org/wiki/INI_file) is a plain text file with
a structure and syntax comprising keyâ€“value pairs organized in sections.

Sections without any key-value pairs defined, however, have no meaning.

## Non-compliant
``` ini
root = true

[*]
end_of_line = crlf
insert_final_newline = true
charset = utf-8

[*.cs]

[*.{cs,json,cshtml,ts}]
indent_style = space
indent_size = 4

[*.{csproj,props,targets,config,nuspec}]
indent_style = space
indent_size = 2
```

## Compliant
``` ini
root = true

[*]
end_of_line = crlf
insert_final_newline = true
charset = utf-8

[*.{cs,json,cshtml,ts}]
indent_style = space
indent_size = 4

[*.{csproj,props,targets,config,nuspec}]
indent_style = space
indent_size = 2
```
